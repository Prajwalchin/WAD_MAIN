FE First Year

  • Engineering Mathematics-I

    Calculus, matrices, and differential equations for engineering analysis.

  • Engineering Physics

    Mechanics, optics, and modern physics concepts with practical applications.

  • Basic Electrical Engineering

    Electrical circuits, network theorems, and basic machine fundamentals.

  • Programming Fundamentals

    Problem solving and C programming basics for computational thinking.

SE Second Year

  • Data Structures

    Linear and non-linear structures for efficient data organization.

  • Object Oriented Programming

    Classes, inheritance, polymorphism, and abstraction in software design.

  • Computer Graphics

    2D/3D transformations, rendering basics, and graphical algorithms.

  • Digital Electronics

    Logic design, combinational and sequential circuits, and digital systems.

TE Third Year

  • Database Management Systems

    Relational models, SQL, transactions, and schema normalization.

  • Operating Systems

    Processes, scheduling, memory management, and synchronization.

  • Computer Networks

    Network layers, routing, transport protocols, and communication models.

  • Web Technologies

    Core frontend-backend concepts for building web applications.

BE Final Year

  • Artificial Intelligence

    Search, learning models, and intelligent systems for real-world problems.

  • Cloud Computing

    Virtualization, cloud services, scalable deployment, and architecture.

  • Cyber Security

    Threat models, cryptography basics, and secure system practices.

  • Project Work

    Capstone engineering project with implementation and documentation.
